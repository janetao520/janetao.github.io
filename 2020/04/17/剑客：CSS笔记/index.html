    CSS简介

    层叠样式表(英文全称：Cascading Style Sheets)是一种用来表现HTML（标准通用标记语言的一个应用）或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。CSS不仅可以静态地修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。
    CSS 能够对网页中元素位置的排版进行像素级精确控制，支持几乎所有的字体字号样式，拥有对网页对象和模型样式编辑的能力。<—-来源：百度介绍>

    1、CSS基础

    1.1：CSS基础及语法

    1.1.1：CSS概述

    CSS指的是层叠样式表
CSS样式表极大地提高了工作效率

    1.1.2：基础语法

    selector[元素]{
    property: value[属性值]
}
例子：h1{color:red; font-size:140px;}
** 属性值大于1个之后，属性之间用分号隔开
** 如果值大于一个单词，则需要加上引号
        p{font-family:"sans serif"}
**DEMO
    引入CSS样式表
   <link href="路径/CSS样式表名" type="text/css" rel="stylesheet">

    1.1.3：CSS高级语法

    1.1.3.1：选择器分组

    例如：h1,h2,h3,h4,h5,h6{color:red}

    1.1.3.2：继承

    例如：body{color:green}

    1.2：CSS基础-派生选择器

    ** 派生选择器
通过依据元素在其位置的上下文关系来定义样式
DEMO:
    给 <li> 标签下的 strong 标签添加样式
    li strong{color:green}

    1.3：CSS基础-id选择器

    1.3.1：ID选择器

    - ID选择器可以为标有 ID 的HTML元素指定特定的样式
- ID选择器以 "#" 来定义
** DEMO : #IDtest{color:red}

    1.3.2：ID选择器和派生选择器

    目前比较常用的方式是 ID 选择器 常常用于建立派生选择器

    1.4：CSS基础-类选择器

    *** 类选择器 ***
- 类选择器以一个点选择 ".divclass"
- class 也可以用作派生选择器

    1.5：CSS基础-属性选择器

    1.5.1：属性选择器

    ** 对带有指定属性的HTML元素设置样式
DEMO：
    定义一个属性，然后引用该属性
    <style type="text/css">
        [属性名 title]{color:blue}
       [title = te]{color:red}
    </style>
    <p title="t">属性选择器</p>
    <p title="te">属性和值选择器</p>
注意：如果在定义属性指定值，引用的时候必须指定
    IE6或更高的版本不支持
