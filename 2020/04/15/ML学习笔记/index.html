    HTML简介

    什么是HTML？

    -HTML是描述网页的一种语言
    -HTML是指超文本标记语言（Hyper text markup language）
    -HTML不是编程语言，是一种标记语言

    HTML5：基础详解

    1、声明 <!DOCTYPE>

    -1：HTML 有不同的版本，只有完全页面中使用的确切HTML版本，浏览器才能完全正确地显示出HTML页面，这就是<!DOCTYPE>
-2：HTML5 ：<!DOCTYPE>
-3：HTML4.01： 
<！DOCTYPE HTML PUBLIC"-//W3C//DTD HTML 4.01 ......>
-4：xHTML1.0： 
<！DOCTYPE HTML PUBLIC"-//W3C//DTD XHTML 1.0 ......>

    2、HTML基础标签

    head(头部标签)    、 body(页面显示部分)

    3、HTML标题

    <h1>.....<h6> 标签进行定义标题

    4、段落

    <p></p> 标签定义段落

    5、链接

    <a> 标签定义链接
例如：<a href="https://www.baidu.com"> 百度 </a>

    6、图像

    <img> 标签定义图像
例如：<img src="images/111.jpg">
      <img src="图片路径">

    HTML5：元素简介及使用方法

    1、元素指的是从开始标签到结束标签夫人所有代码

    2、HTML元素语法

    1：元素的内容是开始标签与结束标签之间的内容
2：空元素在开始标签中进行关闭
3：大多数HTML元素可拥有属性

    3、嵌套的HTML元素

    大多数的HTML元素是可以嵌套的

    HTML5：属性的使用

    1、HTML的属性

    1：标签可以拥有属性为元素提供更多的信息

    2：属性以键/值对的形式出现

    如：href="www.baidu.com"

    3：常用标签属性

    <h1> : align 对齐方式
     <h1>：默认居左
     align="right" 居左
     align="center" 居中
<body>：bgcolor 背景颜色
        background 背景图片
      例如：<body bgcolor="#000"></body>
           <body background="图片路径"></body>
<a>:target 规定在何处打开链接

    4：通用属性

    class :规定元素的类名
id: 规定元素唯一ID
style: 规定元素的样式
title ： 规定元素的额外信息

    HTML5 ：格式化及使用(字体)

    HTML5：样式的使用

    1、标签

    <style> :样式定义
<link>：资源引用

    2、属性

    rel="stylesheet"  外部样式表
type="text/css"  引入文档的类型
margin-left：边距

    3、 HTML样式 ：三种样式表插入方法

    3.1：外部样式表
    例如：<link rel="stylesheet" type="text/css" herf="mystyle.css">
    3.2：内部样式表
    例如：<style type="text/css">
         body{background-color:red}
         p{margin-left:200px}
    </style>
    3.3：内联样式表
    例如：<p style="color:red">

    HTML5、链接属性及使用

    1：链接数据

    文本链接
    例如：<a href="http://www.baidu.com">百度</a>
图片链接
    例如：<a href="http://www.baidu.com"><img src="图片路径"></a>

    2：属性

    href属性：指向另一个文档的链接
name属性：创建文档内的链接 （锚点）
    例如： <a name="tips"></a>
           <a href="#tips"></a>

    3：img标签属性

    alt：替换文本属性
    例如：<img src="xxxxx" alt="替换文本">
 width： 表示 宽
 height：表示 高
     例如：<img src="xxxxx" width="120px" height="120px">

    1:表格使用(代码格式)如下：

    1.1：无边框表格

    1.2：带边框表格

    2:属性

    border ：边框
例如：<table border="2">

    3：定义表行 表头 表格

    tr：定义 行(单独使用不起作用)
th :定义 表头  （起加粗 居中效果）
td：定义表格 （表格数据）
例如：<tr><th>表头 居中 加粗</th></tr>
      <tr><td>单元格1</td><td>单元格2</td><td>单元格3</td></tr>    
      上述代码表示 一行中有三个单元格

    4：caption 居于表格重要部分 ，用于 加粗

    例如 ：<caption></caption>

    5：表格内的标签

    表格里面可以嵌套其他标签
如：<ul>

    6：单元格边距

    cellpadding : 表示单元格边距
例如：<table border="2" cellpadding="100%"></table>

    7：单元格间距

    cellspacing :表示单元格间距
例如：<table border="2" cellspacing="100%"></table>

    8：表格内的背景颜色和图像

    bgcolor :表示单元格颜色
background ：表示单元格里图片
例如：<table border="2" bgcolor=#rrggbb></table>
      <table border="2" background="图片路径"></table>

    9：单元格里面的内容的排列

    10：跨列和跨行

    colspan：表示跨列
rowspan：表示跨行
例如：<td colspan="2"> 表格向下跨两列</td>
      <td rowspan="2"> 表格向右跨两行</td>

    HTML5、列表的使用

    1：列表标签介绍

    2：无序列表

    使用标签 : <ul><li>列表1</li></ul>
属性： disc :实体圆 、 circle ：空心圆 、 square ：方块
例如： <ul type="disc">

    3：有序列表

    使用标签 <ol><li></li></ol>
属性：A 、a、1、i、st 、art
例如： <ol type="A">

    4：嵌套列表

    使用标签：<ul><ol><li>列表</li></ol></ul>
例如 ：<ul>
        <ol>
            <li>1</li> <li>2</li>   
        </ol>
        </ul>

    5：自定义列表

    使用标签：<dl>、<dt>、<dd>
    例如：<dl>
                <dt>主标题</dt>
                <dd>副标题</dd>
          </dl>

    HTML5、块元素标签的使用

    1：HTML块元素

    块元素在显示时，通常会以新行开始（块元素占一行）
如：<h1>、<p>、<ul>

    2：HTML内联元素

    内联元素通常不会以新行开始
如：<b>、<a>、<img>

    3：HTML -div元素

    <div>元素也被称为块元素，其主要是组合HTML元素的容器

    4：HTML -span元素

    <span> 元素是内联元素，可作为文本的容器

    HTML5、XHTML的规范化

    1:XHTML简介

    1：什么是XHTML

    XHTML指的是可扩展超文本标记语言
XHTML与HTML4.01几乎是相同的（指的是语法和书写形式）
XHTML是更严格更纯净的HTML版本  
XHTML是以XML应用的方式定义的HTML
XHTML得到所有主流浏览器的支持

    2：为什么使用XHTML

    为了代码的完整性和良好性

    3：文档声明

    DTD：规定了使用通用标记语言的网页语法

    4：三种XHTML文档类型

    STRICT (严格类型)
TRANSITIONAL (过渡类型)
FRAMSET (框架类型)

    2：元素

    元素语法
1：必须正确嵌套
2：必须始终关闭
3：必须小写
4：必须有一个根元素

    3：属性

    1：必须使用小写
2：必须用引用号包围
3：最小化也是禁止的
