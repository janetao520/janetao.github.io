    HTML简介

    什么是HTML？

    -HTML是描述网页的一种语言
-HTML是指超文本标记语言（Hyper text markup language）
-HTML不是编程语言，是一种标记语言

    HTML5：基础详解

    1、声明 <!DOCTYPE>

    -1：HTML 有不同的版本，只有完全页面中使用的确切HTML版本，浏览器才能完全正确地显示出HTML页面，这就是<!DOCTYPE>
-2：HTML5 ：<!DOCTYPE>
-3：HTML4.01： 
<！DOCTYPE HTML PUBLIC"-//W3C//DTD HTML 4.01 ......>
-4：xHTML1.0： 
<！DOCTYPE HTML PUBLIC"-//W3C//DTD XHTML 1.0 ......>

    2、HTML基础标签

    head(头部标签)    、 body(页面显示部分)

    3、HTML标题

    <h1>.....<h6> 标签进行定义标题

    4、段落

    <p></p> 标签定义段落

    5、链接

    <a> 标签定义链接
例如：<a href="https://www.baidu.com"> 百度 </a>

    6、图像

    <img> 标签定义图像
例如：<img src="images/111.jpg">
      <img src="图片路径">

    HTML5：元素简介及使用方法

    1、元素指的是从开始标签到结束标签夫人所有代码

    2、HTML元素语法

    1：元素的内容是开始标签与结束标签之间的内容
2：空元素在开始标签中进行关闭
3：大多数HTML元素可拥有属性

    3、嵌套的HTML元素

    大多数的HTML元素是可以嵌套的

    HTML5：属性的使用

    1、HTML的属性

    1：标签可以拥有属性为元素提供更多的信息

    2：属性以键/值对的形式出现

    如：href="www.baidu.com"

    3：常用标签属性

    <h1> : align 对齐方式
     <h1>：默认居左
     align="right" 居左
     align="center" 居中
<body>：bgcolor 背景颜色
        background 背景图片
      例如：<body bgcolor="#000"></body>
           <body background="图片路径"></body>
<a>:target 规定在何处打开链接

    4：通用属性

    class :规定元素的类名
id: 规定元素唯一ID
style: 规定元素的样式
title ： 规定元素的额外信息

    HTML5 ：格式化及使用(字体)

    HTML5：样式的使用

    1、标签

    <style> :样式定义
<link>：资源引用

    2、属性

    rel="stylesheet"  外部样式表
type="text/css"  引入文档的类型
margin-left：边距

    3、 HTML样式 ：三种样式表插入方法

    3.1：外部样式表
    例如：<link rel="stylesheet" type="text/css" herf="mystyle.css">
    3.2：内部样式表
    例如：<style type="text/css">
         body{background-color:red}
         p{margin-left:200px}
    </style>
    3.3：内联样式表
    例如：<p style="color:red">

    HTML5、链接属性及使用

    1：链接数据

    文本链接
    例如：<a href="http://www.baidu.com">百度</a>
图片链接
    例如：<a href="http://www.baidu.com"><img src="图片路径"></a>

    2：属性

    href属性：指向另一个文档的链接
name属性：创建文档内的链接 （锚点）
    例如： <a name="tips"></a>
           <a href="#tips"></a>

    3：img标签属性

    alt：替换文本属性
    例如：<img src="xxxxx" alt="替换文本">
 width： 表示 宽
 height：表示 高
     例如：<img src="xxxxx" width="120px" height="120px">
